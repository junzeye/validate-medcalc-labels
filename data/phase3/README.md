# Phase 3 MD Blind Evaluation

Both Excel files contain:
- All labels generated by the Phase 2 recomputation workflow, only selecting ones with supermajority agreement in 5 independent pipeline outputs. These correspond to the `y_new` column in each of the two Excel files.
- 50 sampled labels independently computed by three licensed physicians at Stanford (Daniel Tawfik, Alexander Goodell, and Nikhil Kotha; respectively, specialities in pediatrics, perioperative medicine, and radiation oncology). These correspond to the `y* (Physician label)` column.

Rows with empty physician label fields indicate samples that were not reviewed by physicians—we consolidated everything into single files for convenience.

## Column Naming

| Column | Description |
|--------|-------------|
| `y_orig` | $\hat{y}_{\textsf{original}}$ |
| `y_new` | $\hat{y}_{\textsf{new}}$ |
| `y* (Physician label)` | Physician label (if non-empty) |
| `y_final` | Maintained test labels after incorporating physicians' feedback (case study output) |
